# 銀河科技

一個 Slimefun 附加插件, 它添加了銀河系, 恆星系統, 星球, 月球, 火箭, 與太空服. 它有一個用於附加開發者的大型api.

靈感來自 ClayTech 與模組 GalactiCraft.
